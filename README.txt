README

Quick and easy way for tabaholics to close and save their tabs for later.

Ctrl+Q - Close the current tab and Chrome will save it for you.
Ctrl+M - Open a new window with all your saved tabs.
Ctrl+B - Close that magic window and Chrome will save the tabs for you again. 

All the tabs are stored in a folder in your bookmarks called "Saved Tabs", usually under your "Other Bookmarks"
